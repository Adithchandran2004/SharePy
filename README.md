##


### Installation

- Just, Clone this repository -
  ```
  git clone --depth=1 ezhuthu
  ```

- Now go to cloned directory and install requirements` -
  ```
  $ pip install -r requirements.txt
  ```
- or(manually)`-
  ```
  $ pip install tqdm
  $ pip install colorama
  ```
- Now run the main.py file`-
  ```
  $ cd SharePy
  $ python3 main.py
  ```
